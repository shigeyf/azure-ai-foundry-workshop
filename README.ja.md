# Azure AI Foundry ワークショップ: 初心者から中級者に向けて

[English](./README.md) | [日本語](./README.ja.md)

ローカルコンピュータで学習する際は、VSCode ワークスペースファイル [`default`](./default.code-workspace) を開いてください。
